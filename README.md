# HDL Copilot

VSCode plugin backed by a C++ LSP server.

https://marketplace.visualstudio.com/items?itemName=metalware-inc.hdl-copilot

## Building

```
cd server
./build.sh
```

## Unit tests

```
./build/tests/hdl_copilot_server_tests
```

## Integration tests

From the `tests` directory:

```
./run_integration_tests.sh
```
